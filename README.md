# Hubot Quizz French

> hubot: quizzfr
> hubot: score

## Install

Add hubot-quizzfr to your *package.json*.

    npm install --save hubot-quizzfr

Then add hubot-quizzfr to  *external_scripts.json*.

```javascript
[..., "hubot-quizzfr", ...]
```
